Сначала клонируйте репозиторий в свободную папку:

Выполните команду git clone https://github.com/sacist/TestLanding.git

Перейдите в каталог проекта:

cd TestLanding

Установите все зависимости:

npm install

Запустите проект:

npm run dev

После этого откройте браузер и перейдите на http://localhost:5173/

Должно всё сработать
